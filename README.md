# SvelteKit + LayerCake (powered by D3) playground

Issue context (in french): stephane-klein/backlog#311

This playground uses: https://layercake.graphics/

## Getting started

```sh
$ asdf install
```

```sh
$ pnpm install
```

Start web server:

```sh
$ pnpm run dev
```

Go to:

- http://localhost:5173/page1
- http://localhost:5173/page2
